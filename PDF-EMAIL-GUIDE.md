### How to Generate PDF

When submitting a form, you have two options:

1. **Generate PDF Document** - Creates a PDF file of the complete form
2. **Email PDF to Administrator** - Generates PDF and prepares it for email

### Steps to Generate PDF

1. **Fill out the form** completely
2. **Scroll to "PDF Generation & Email Options"** section (before submission)
3. **Check "Generate PDF Document"** checkbox
4. **Click "Submit & Generate Tree"**
5. **PDF is automatically generated** and saved to your Downloads folder

### PDF File Details

- **File Name Format**: `Genealogy-Form-{FormNumber}-{Date}.pdf`
- **Example**: `Genealogy-Form-5-2024-01-15.pdf`
- **Location**: Your Downloads folder
- **Content**: Complete form with all sections and data

### PDF Contents

The generated PDF includes:
- Form title and metadata (Form #, Version)
- Section 1: Family Identification
- Section 2: Earliest Known Ancestor (with all details)
- Section 3: Generation-wise Lineage (all generations and children)
- Section 5: Scholarly Contributions (if filled)
- Section 11: Compiler Details
- Page numbers and generation date

## ðŸ”§ Troubleshooting

### PDF Not Generating

**Issue:** PDF doesn't download  
**Solution:**
- Check browser popup blocker settings
- Allow downloads for this site
- Try using Chrome or Firefox browser
- Check Downloads folder permissions

### Email Client Not Opening

**Issue:** Clicking "Open Email Client" doesn't work  
**Solution:**
- Make sure you have an email client installed
- For webmail (Gmail, Outlook.com), manually compose email
- Copy the email address: **abdinzaghi@gmail.com**
- Attach the PDF manually

### PDF File Not Found

**Issue:** Can't find the PDF file  
**Solution:**
- Check your Downloads folder
- File name: `Genealogy-Form-{Number}-{Date}.pdf`
- Search for "Genealogy-Form" in Downloads
- Check browser download history

### Email Attachment Too Large

**Issue:** Email service rejects attachment  
**Solution:**
- Most email services support up to 25MB
- PDF files are typically small (< 1MB)
- If issue persists, use cloud storage (Google Drive, Dropbox)
- Share the link via email instead
